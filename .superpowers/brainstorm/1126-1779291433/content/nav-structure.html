Where should the new features live?

We're adding: Budgets, Savings Goals, Search, Recurring markers, Date range filter. The current nav has 5 tabs — we need to decide how to absorb the new features.

A

Add 2 new tabs: Budgets + Goals

Nav becomes: Dashboard · Transactions · Reports · Budgets · Goals
Categories + Settings move into a "More" overflow or Settings page.
Pro: Budgets and Goals are first-class. Con: 5+ tabs gets crowded on mobile.

B

One new tab: Budget & Goals combined

Nav becomes: Dashboard · Transactions · Budget · Reports · Settings
A single "Budget" page handles both spending limits and savings goals.
Pro: Clean 5-tab nav, no overflow. Con: Two distinct concepts in one page.

C

Integrate into existing pages

Budgets live inside Categories (set a limit per category). Goals live on Dashboard as a new section. Search + filters added to Transactions page.
Pro: No nav changes needed. Con: Features harder to discover.
